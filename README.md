# 🛡️ MicroHedge

**MicroHedge** is a **decentralized commodity hedging platform** built for the real world — **powered by Injective**.  
It brings **simple and accessible hedging for farmers and small businesses** to protect against price volatility in commodities like coffee, corn, etc.  
[Live App](https://micro-hedge.vercel.app)

---

## 🚀 Overview

Across emerging markets, farmers and small enterprises suffer from **unstable commodity prices**:

- ☕ Coffee prices drop → income loss  
- 🌽 Corn prices fluctuate → planning becomes hard  
- ❌ Traditional tools are complex, expensive, and inaccessible  

**MicroHedge solves this by making DeFi hedging intuitive, affordable, and practical**, with **AI-powered hedge recommendations already integrated**.

---

## 🛠 How It Works

1. **Connect Wallet** – Users connect their wallet on the web app.  
2. **Select Commodity** – Choose a commodity (coffee, corn, etc.).  
3. **Set Terms** – Input amount, strike price, and expiry date.  
4. **Lock Funds in USDT** – Users lock USDT (mock USDT on Injective Testnet for testing).  
5. **AI Hedge Recommendation** – Integrated AI suggests optimal hedge parameters.  
6. **Smart Contract Logic** – Smart contracts manage hedges automatically.  
7. **Settlement**:  At expiry, the Price Oracle provides the real market price:
   - If market price < strike → user is protected  
   - If market price > strike → user benefits from upside  

Everything runs seamlessly on **Injective**, providing fast, low-fee, and secure transactions.
